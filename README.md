# Hospital-Pharmacy-PowerBI-Analysis

## Project Overview
This Power BI project analyzes patient visits, disease patterns, and pharmacy expenditure within a county referral hospital.

## Repository Structure
- PowerBI_Report → Final Power BI (.pbix) file
- Report_Summary → Executive summary report
- Screenshots → Dashboard preview image

## Key Insights
- Infectious and chronic diseases dominate visit volume
- Pharmacy costs are influenced by treatment intensity, not just volume
- Seniors consume the highest medication quantity
